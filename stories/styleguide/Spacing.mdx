# Spacing

Spacing plays a vital role in ensuring visual harmony in a design system. It maintains consistency and improves user experience by providing a balanced layout. Our design system uses a modular scale for spacing, which helps create meaningful and predictable gaps between components.

## Spacing Scale

We have a range of predefined spacing units based on a multiple scale. These units allow designers and developers to apply consistent spacing values throughout the user interface.

### Base Unit

Our base unit of spacing is `4px`. All other spacing values are multiples of this base unit.

## Spacing Tokens

Design tokens are specific values that can be referenced in code. These spacing tokens are used in CSS variables or JavaScript objects to maintain consistency across the product.
