export const ColorBlock = ({ colorHex, colorHsl, name }) => {
  const copyToClipboard = (value) => {
    navigator.clipboard.writeText(value);
    alert(`Copied ${value} to clipboard!`);
  };

  return (
    <article className='flex justify-center flex-col items-center gap-4'>
      <div
        className="color-block"
        style={{ backgroundColor: colorHex }}
        onClick={() => copyToClipboard(colorHex)}></div>
      <span className="flex flex-col gap-1 items-center text-xs">
        <span className='font-bold'>{name}</span>
        <span className='inline-flex gap-1'>
          {colorHex}
          <button
            onClick={(e) => {
              e.stopPropagation(); // Prevent double copying

              //copyToClipboard(colorHex);
            }}
          >
            <i className="bi bi-copy"></i>
          </button>

  );

# Colors

Colors play a vital role in a design system, helping establish a consistent visual identity, communicate information, and enhance the user experience. Our color palette is designed to balance aesthetic appeal with functionality, providing the foundation for creating accessible and intuitive interfaces.
A comprehensive palette for the design system, providing HEX and HSL values for each color group, with a copy-to-clipboard option.

## Brand Primary Colors

### What is brand color?

The brand or primary color represents the core identity of our brand. It reflects our values and is typically the most recognizable aspect of the brand’s visual presence. Our brand color is used across key interface elements to create a cohesive look and feel and to reinforce brand recognition.

<Alert title="Usage" message="The brand color should be used for primary actions, call-to-action buttons, and critical points of interaction. It ensures that the brand identity remains strong and memorable across all platforms and applications." />

### Secondary

### What is a Secondary Color?

The secondary color complements the primary color and is often used to highlight secondary actions or features in the user interface. Secondary colors should provide enough contrast with the primary color to offer a visual hierarchy without overpowering it.

Accent colors are used sparingly in the design system to bring attention to specific details or interactive elements. They add vibrancy and can enhance the UI without overwhelming it, often used for highlights or special UI features.

## Semantic Colors

Semantic colors convey meaning and feedback in a user interface, helping users recognize and respond to information quickly. These colors are especially useful for signaling statuses like success, error, warning, and information.

## Natural Colors
Neutral colors are foundational to any design system and are primarily used for typography, backgrounds, borders, and other interface elements that do not require emphasis. Neutral colors help ensure readability and give prominence to primary and accent colors by providing a balanced background.
